# From Trade Log to Verdict — the statistics of a significant edge

This tutorial is maintained as **one self-contained HTML page**, so its design and content
can't drift across two files.

- **Read it (rendered):** <https://mspinola.github.io/crucible/>
- **Source:** [`docs/index.html`](index.html)

It covers everything from the R-multiple trade log through the full gauntlet
(`REAL → STRONG → DURABLE → GENERAL`), worked end to end on a Donchian breakout — the
runnable version is [`examples/donchian_gauntlet.py`](../examples/donchian_gauntlet.py).
